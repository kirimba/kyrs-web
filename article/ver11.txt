<h2>Internet Explorer 11</h2>
<p>одиннадцатая, и последняя версия обозревателя Internet Explorer от Microsoft и преемник Internet Explorer 10. Поставляется в составе Windows 8.1 и Windows Server 2012 R2, Windows 10 и Windows Server 2016.</p>
<p>Internet Explorer 11 вышел в обновлении Windows 8.1, которое было представлено 17 октября 2013 года. На данный момент Internet Explorer 11 доступен для Windows 8.1, Windows 8.1 с обновлением, Windows 7 и Windows Phone 8.1. Браузер поддерживает синхронизацию открытых вкладок, настроек, паролей и избранного при использовании учетной записи Microsoft.</p>
<p>Этот релиз включил в себя улучшенное масштабирование для экранов с большим расширением, предварительную загрузку HTML5, перемещение элементов мышью, аппаратно ускоренное декодирование изображений формата JPEG, и полноэкранный режим HTML5. Internet Explorer 11 является первой версией, поддерживающей WebGL555657 и протокол SPDY (начиная с версии 3).</p>
<p>Microsoft заявила, что Internet Explorer 11 при прохождении теста для измерения исполнительности JavaScript SunSpider является самым быстрым браузером для Windows по состоянию на 15 октября 2013 года.</p>
<p>Internet Explorer 11 включил себя специальные функции для работы на мобильных устройствах. Среди них выполнение поиска из адресной строки, режим чтения, который позволяет фокусироваться только на основном контенте веб-страницы, большой размер вкладок для удобства переключения между ними на сенсорных устройствах, возможность закрепления любимых сайтов на начальном экране и одновременной работы с приложениями и браузером.</p>
<p>Для обеспечения совместимости с предыдущими веб-приложениями в Internet Explorer 11 для Windows 8.1 и Windows 7 встроен Режим Предприятия (Enterprise Mode).</p>
<p>Internet Explorer 11 — первая версия Internet Explorer, в которой пропала возможность отключать вкладки.</p>
<p>Internet Explorer 11 имеет интерфейс, который автоматически адаптируется под тип устройства — например, на планшете на Windows 8 он обеспечит лучшие возможности для сенсорного управления (вкладки по размеру пальца, пролистывание и т. д.), в то время как для ноутбука будет поддерживать управление с помощью мыши и клавиатуры.</p>
<p>Internet Explorer 11 — последний браузер в семействе Internet Explorer. Начиная с Windows 10, ему пришёл на замену новый браузер <i><b>Microsoft Edge</b></i>.</p>
